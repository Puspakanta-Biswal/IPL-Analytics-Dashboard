# 🏏 IPL Analytics Dashboard (Power BI)

## 📊 Project Overview
This project delivers an end-to-end data analytics solution on IPL match data, transforming raw data into actionable insights on team performance, toss impact, and venue trends.

---

## 🎯 Business Problem
Analyze how match outcomes are influenced by:
- Toss decisions (Bat vs Field)
- Team performance over seasons
- Venue-based advantages

---

## 🔑 Key Features
- 📌 KPI Cards: Matches, Teams, Toss Impact %, Venues
- 📊 Interactive Dashboard with slicers (Season, Team, Venue)
- 🔄 Data Cleaning & Transformation using Power Query
- 🧠 Data Modeling & DAX Calculations
- 📈 Insight-driven visual storytelling

---

## 🛠 Tools & Technologies
- Power BI (Dashboard Development)
- Jupyter Notebook (Data Cleaning & Exploration)
- Python (Pandas, NumPy)
- Excel / CSV Dataset
- DAX (Data Analysis Expressions)

## 🔍 Key Insights
- Toss winning teams have ~52% higher probability of winning
- Fielding first is the most preferred toss decision
- Certain venues show strong bias toward chasing teams
- Top teams consistently outperform across seasons
